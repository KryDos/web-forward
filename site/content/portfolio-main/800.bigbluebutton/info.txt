info: BigBlueButton Inc. is a not-for-profit organization dedicated to providing remote students a high-quality learning experience through BigBlueButton, an open source web conferencing system for distance education. Using BigBlueButton, remote students can interact with teachers through real-time sharing of audio, video, chat, presentations, and desktops. BigBlueButton records sessions for later viewing and uses <a href="http://popcornjs.org/" target="_blank">popcorn.js</a> for playback in HTML5.
